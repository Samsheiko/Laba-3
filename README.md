## Задание 1







### Задача 1







### Текст задачи







#### На основе заданного списка чисел получить список из значений true и false. True
— если соответствующий элемент исходного списка чётный, и false — в
противном случае. Seq.map






### Алгоритм решения



######

##### Вот краткие шаги его работы:Запрос ввода: Программа просит пользователя ввести числа через пробел.Чтение строки: Метод Console.ReadLine() считывает всю строку целиком.Разделение (Токенизация): Строка разбивается по символу пробела на отдельные текстовые элементы (строковые числа).Преобразование типов: Текстовые элементы поочередно конвертируются в целые числа (int).Проверка на четность: Каждое число делится на 2. Если остаток равен 0, элемент заменяется на true, иначе — на false.

# Задание 2







###



### Текст задачи







#### Список содержит строки, обозначающие римские числа от I до IX. Найти в
десятичной системе счисления сумму элементов списка.







### Алгоритм решения







#### Запрос ввода: Программа просит пользователя ввести римские числа.Очистка и разделение:Считывает строку и переводит её в нижний регистр (.ToLower()).Разбивает строку по пробелам, игнорируя лишние повторяющиеся пробелы (RemoveEmptyEntries).Конвертация (Сопоставление): Функция toInt проверяет каждое слово. Если это римская цифра от i до ix, она заменяет её на обычное число (1–9). Любые другие символы превращаются в 0.Слепое суммирование (Свёртка): Метод Seq.fold проходит по всем элементам, берет их числовые значения и поочередно прибавляет к общему счетчику, который изначально равен 0.Вывод результата: Программа печатает на экране итоговую сумму.
