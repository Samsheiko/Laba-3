## Задание 1







### Задача 1







### Текст задачи







#### На основе заданного списка чисел получить список из значений true и false. True
— если соответствующий элемент исходного списка чётный, и false — в
противном случае. Seq.map






### Алгоритм решения



######

##### Программа запрашивает количество чисел, лениво считывает их в последовательность, преобразует в true/false по чётности через Seq.map и выводит результат, вычисляя значения «на лету» при приведении к списку.

# Задание 2







###



### Текст задачи







#### Список содержит строки, обозначающие римские числа от I до IX. Найти в
десятичной системе счисления сумму элементов списка.







### Алгоритм решения







#### Программа запрашивает количество чисел, лениво считывает римские строки через seq { ... } в верхнем регистре, с помощью Seq.fold пошагово конвертирует их в десятичную систему и суммирует с нуля, после чего выводит готовый результат на экран.
